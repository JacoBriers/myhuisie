# Huisie - House Cleaning Tracker

A simple, browser-based app to track house cleaning tasks with timers.

## Features

- **Rooms & Tasks**: Default rooms (Kitchen, Lounge, Dining Room, Ensuite Bathroom, Main Bedroom, Bedroom 1, Bedroom 2, Playroom, Bathroom, Toilet) each with tasks (Tidy, Vacuum/Mop, Move Furniture & Vacuum, Change Bedding)
- **Timers**: Each task has a target timer (default 10 minutes), adjustable in 1-minute increments
- **Completion**: Satisfying tick animation/sound when marking a task complete
- **Beep**: Single beep when a timer runs out
- **Custom rooms & tasks**: Add or delete rooms and tasks
- **Persistent state**: All changes (rooms, tasks, timers, completion status) saved to browser localStorage

## Usage

Open `index.html` in a browser, or visit the GitHub Pages URL.

## Deployment

This is a static site — no build step required. Just push to the `main` branch and enable GitHub Pages in repository settings (source: root).
